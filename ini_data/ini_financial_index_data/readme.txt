Outstanding 2 (HK)(1)&Outstanding 2 (HK)(2):用于计算Market Capitalization.这两个文件要合并一下，各个股票每年6月和12月分别提取了outstanding shares的数目，乘以年报当天的价格就是Market Capitalization.
Book Value (HK)：Book value of equity.
BPS (HK)：对应Word power文章中的BM属性.每股净资产BPS=(归属母公司股东的权益-优先股)／报告期末总股本
EPS Basic (HK)：每股收益。用年报当天的股价除以每股收益能够得到PE ratio
Equity Multiplier (HK)：权益乘数＝1／（1-资产负债率）=期末资产总计/期末所有者权益合计
ROE Average (HK)：ROE(平均）＝归属母公司股东净利润／[（期初归属母公司股东的权益＋期末归属母公司股东的权益）／2]*100%。注：平均归属母公司股东的权益为负的不进行计算。
ROE Diluted (HK)：ROE(摊薄）＝归属于母公司股东的净利润／期末归属于母公司所有者权益合计*100%
基本每股收益是指企业按照归属于普通股股东的当期净利润，除以发行在外普通股的加权平均数计算出的每股收益。
